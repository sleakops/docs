# Connect with VPN

To handle VPN connection we use Pritunl. You can download the client [here](https://client.pritunl.com/#install).

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/VPN-pritunl-client.png" alt="vpn-pritunl-client" />
</Zoom>

Once you have created a [Provider ](/provider/index.mdx), go to [SleakOps dashboard](https://console.sleakops.com/) and select the to which account you want to get VPN access. Remember that to do this you need to have access to the VPN of that account but also the VPN should be created, we create the VPN of a specific account when the first cluster of that account is created. 

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/VPN-account-select.png" alt="vpn-account-select" />
</Zoom>

This will prompt you with what is called the URI profile, it has validation for 24 hours and you have to load it into the Pritunl client.

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/VPN-uri-profile.png" alt="vpn-uri-profile" />
</Zoom>

Copy it and import it into the Pritunl client and you'll be able to connect to it:
