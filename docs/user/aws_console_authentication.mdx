# AWS Console Authentication

To log in to the AWS Console for any of the accounts, as described in the [Architecture Overview](/provider/awsprovider/aws), the first step is to be logged into your SleakOps user. you'll have to enter the 'security' account and from there assume the role of the account you want. The easiest way to do this is by using [Sleakops Dashboard](https://console.sleakops.com/):

Press: <Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)"> <img src="/img/users/USER-get-access-button.png" alt="USER-get-access-button" style={{ height: '50px', width: 'auto' }}/></Zoom>
And then press <Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)"> <img src="/img/users/USER-login-button.png" alt="USER-login-button" style={{ height: '50px', width: 'auto' }}/></Zoom>

This will open the AWS login form. The Account ID field should be automatically filled with the 'security' account ID. If this doesn't happen, it might be because another service is attempting to fill the fields.

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img
    src="/img/users/USER-aws-security-login.png"
    alt="USER-aws-security-login"
    style={{ height: "300px", width: "auto" }}
  />
</Zoom>

Once logged into the 'security' account, it will appear as shown in the following image:

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img
    src="/img/users/USER-aws-console-initial.png"
    alt="USER-aws-console-initial"
  />
</Zoom>

Now, in the AWS console, you need to return to the SleakOps dashboard and select the account you want to log in to:

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/USER-account-switcher.png" alt="USER-account-switcher" style={{ height: '150px', width: 'auto' }}/>
</Zoom>

This will prompt a new AWS tab to switch roles into the account you've selected. When you switch accounts, you'll leave the 'security' AWS account and move to the selected one. In this case, I switched to the 'production' account via the 'SleakopsAdminRole' because my user is an Admin, and I'm allowed to assume this role.

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img
    src="/img/users/USER-switch-role.png"
    alt="USER-switch-role"
    style={{ height: "275px", width: "auto" }}
  />
</Zoom>

For more information about this process, you can read its [AWS documentation <FiExternalLink/>](https://docs.aws.amazon.com/IAM/latest/UserGuide/id_roles_use_switch-role-console.html).
