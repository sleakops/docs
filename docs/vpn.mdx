# Connect with VPN

To handle VPN connection we use Pritunl. You can download the client [here](https://client.pritunl.com/#install).

<img src="/img/pritnl.webp" width="400px" />

Once you have created a [Provider ](/provider/index.mdx), go to [SleakOps home](https://console.sleakops.com/) and select the Provider:

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/VPN-provider-select.png" alt="vpn-provider-select" />
</Zoom>
