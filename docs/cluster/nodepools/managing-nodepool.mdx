# Managing a Node Pool

Here you will find: 

- How to update the settings of a node pool
- How to delete a node pool.

:::info

Remember that build node pools can’t be edited or deleted. 

:::

### 1. Access your cluster’s settings to access the Node Pools section

From the *Cluster Listing*, select a node pool and access the *Setting* option. Then, click on the *Node Pools* box.

### 2. Select the Node Pool
