# Getting Started with Clusters

## How Do I Begin?

Una vez que ya tienes creado tu proveedor Cloud el proximo paso es el de crear un Cluster, para ello tienes que utilizar el Sidebar para viajar entre los recursos disponibles a desplegar, situarte en Cluster y proceder con la creacion.
