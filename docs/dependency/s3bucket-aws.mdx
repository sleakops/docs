# AWS S3 Bucket

For storage solutions, Sleakops supports as a [dependency](./index.mdx) Amazon Simple Storage Service (Amazon S3), with the benefits of object storage built to store and retrieve any amount of data from anywhere. You can read more about those benefits in [AWS S3 docs <FiExternalLink/>](https://aws.amazon.com/s3/).

## Parameters

When you create this dependency, you only need to provide the parameters listed below:

| Name         | Description                                                                                                                                                          |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| s3BucketName | Global identifier for S3 Bucket, you can specify only 24 characters.                                                                                                 |
| s3Acl        | Which set of ACL Permissions will have S3, check them on [AWS <FiExternalLink/>](https://docs.aws.amazon.com/AmazonS3/latest/userguide/acl-overview.html#canned-acl) |

The others parameters of this resource are the default given by Pulumi.
