# AWS SQS

Sleakops incorporates Amazon Simple Queue Service (SQS) into your application architecture. 
This fully managed message queuing service allows you to decouple components and enhance the scalability and reliability of your applications, ensuring smooth and efficient message processing.

You can find much more information about *SQS* on its [developer guide <FiExternalLink/>](https://docs.aws.amazon.com/AWSSimpleQueueService/latest/SQSDeveloperGuide/welcome.html).
