# Release

## What is a Release

In Sleakops a release represents a deployable state of all the services (web-services, workers, cronjobs, jobs) of a [project](/project) in a [environment](/environment)

## Release Creation

Sleakops administrate release for you. every you modify, delete or add a servicer, worker, hook or cronjobs. Sleakops give you option to publish change. 
Every time to publish those changes Sleakops create a new release with autoincrement versions.
