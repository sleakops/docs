# Deploy a Next.js App

En este tutorial aprenderás a desplegar tu aplicación Next.js en SleakOps con pipelines de CI/CD automáticos.

## Prerequisitos

- Una cuenta en SleakOps
- Una aplicación Next.js lista para producción
- Un repositorio en GitHub, GitLab o Bitbucket

## Paso 1: Crear un nuevo proyecto

1. Ingresa a la [consola de SleakOps](https://console.sleakops.com)
2. Haz clic en "Nuevo Proyecto"
3. Selecciona tu repositorio

## Paso 2: Configurar el build

```yaml
# sleakops.yaml
build:
  dockerfile: Dockerfile
  context: .
```

## Paso 3: Desplegar

Una vez configurado, cada push a tu rama principal activará un nuevo despliegue automáticamente.
