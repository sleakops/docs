## 🚀 New Features

- **Domain Validation Button**  
A "check validation" button has been added to the domain drawer for easier domain management.
- **Activity Log Table**  
An activity log table has been created.
- **Access Key Encryption**  
Access keys for code version providers (GIT) are now encrypted.
