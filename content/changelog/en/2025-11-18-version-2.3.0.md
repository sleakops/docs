`minor release` · 11/18/2025

## 🚀 New Features

- **Environment Cloning**  
New function to clone Environments.
- **Project Cloning**  
New function to clone Projects.
- **Dependency Cloning**  
New function to clone Dependencies.
- **Workload Cloning**  
New function to clone Workloads.
- **Variable Group Cloning**  
New function to clone Variable Groups.
- **Cluster Monitoring Filters**  
Added filters for navigating Cluster events in Cluster Monitoring.
- **Variable Group Search**  
Ability to search Variable Groups by internal key names from the general search.

---

## 🐛 Bug Fixes

- **Selected Account**  
The selected account now persists when switching users.
- **Web Services Shutdown**  
Improved Web Services shutdown.
