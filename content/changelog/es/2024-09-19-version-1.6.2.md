## 🚀 Nuevas funcionalidades

- **Actualizaciones**  
Actualizado **Prometheus**, **Loki**, y **EBS CSI Driver** a las últimas versiones a partir de agosto de 2024.
- **Migración de EBS CSI Driver**  
SleakOps ahora usa el Addon EKS gestionado por AWS para el EBS CSI Driver, reemplazando la versión auto-gestionada.
- **Prometheus y Loki con EBS**  
Prometheus ahora utiliza volúmenes EBS para persistencia de datos, previniendo pérdida de datos incluso si los pods fallan.
- **Loki con SimpleScalable**  
Adopta una estructura **SimpleScalable** con almacenamiento **TSDB** para logs, mejorando el rendimiento.
- **Colas Dead-letter de SQS**  
Ahora soporta la creación de **colas SQS** con colas dead-letter asociadas para mejor manejo de errores.
