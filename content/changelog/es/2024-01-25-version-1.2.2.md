## 🚀 Nuevas funcionalidades

- **Botón de Validación de Dominio**  
Se ha agregado un botón "verificar validación" al drawer de dominio para facilitar la gestión de dominios.
- **Tabla de Log de Actividad**  
Se ha creado una tabla de log de actividad.
- **Encriptación de Access Keys**  
Las access keys para proveedores de versión de código (GIT) ahora están encriptadas.
