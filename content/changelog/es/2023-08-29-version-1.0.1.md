## 🚀 Nuevas funcionalidades

- **Gestión de Suscripciones**  
El login y actualizaciones de token se controlan según el estado de la suscripción. Adicionalmente, se implementó una nueva API para registrar usuarios y empresas, validando suscripciones pendientes, con un nuevo modelo para mejor gestión de suscripciones, integrando AwsClient.
- **Onboarding de Marketplace**  
Proceso simplificado para crear usuarios que vienen de un marketplace.
