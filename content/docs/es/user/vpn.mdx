# Conexión a la VPN

Para manejar las conexiones VPN utilizamos Pritunl. Puedes descargar el cliente [aquí](https://client.pritunl.com/#install).

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/VPN-pritunl-client.png" alt="cliente-vpn-pritunl" style={{ height: '300px', width: 'auto' }}/>
</Zoom>

Una vez que hayas creado un [Proveedor](/docs/provider), ve al [Panel de SleakOps](https://console.sleakops.com/) y selecciona la cuenta para la cual quieres obtener acceso VPN. Recuerda que para hacer esto, necesitas tener acceso a la VPN de esa cuenta, pero la VPN también debe estar creada. Creamos la VPN de una cuenta específica cuando se crea el primer clúster de esa cuenta.

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/USER-get-access.png" alt="vpn-obtener-acceso" style={{ height: '225px', width: 'auto' }}/>
</Zoom>

<Zoom overlayBgColorEnd="rgba(255, 255, 255, 0.8)">
  <img src="/img/users/VPN-get-uri.png" alt="vpn-obtener-uri" />
</Zoom>

Esto te mostrará lo que se denomina perfil URI. Tiene un período de validación de 24 horas, y debes cargarlo en el cliente Pritunl.
