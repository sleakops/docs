Aprende cómo exportar tu perfil de VPN como un archivo OpenVPN `.ovpn` y conectarte usando cualquier cliente compatible con OpenVPN, incluidas apps móviles.

## Prerrequisitos

- Cuenta en SleakOps con acceso VPN configurado
- VPN del Cluster habilitada ([Documentación de VPN](/user/vpn))

## Empecemos

### Obtener el archivo .ovpn

Seguí la [guía de conexión a la VPN](/user/vpn) para obtener tu link de Pritunl. El link tiene esta forma:

```
pritunl://127.0.0.176/ku/bURgBN8W
```

Para acceder a la página de descarga, modificá la URL de la siguiente manera:

- Reemplazá `pritunl://` por `https://`
- Reemplazá `/ku/` por `/k/`

El resultado debe quedar así:

```
https://127.0.0.176/k/bURgBN8W
```

Abrí esa URL en el navegador. Verás una página con opciones de descarga:

{/* TODO: screenshot - Página de Pritunl mostrando los botones "Download Profiles (tar)" y "Download Profiles (zip)" */}

Hacé clic en **Download Profiles (tar)** o **Download Profiles (zip)**, luego extraé el archivo para obtener el `.ovpn`.

### Configurar OpenVPN en el teléfono

Descargá la app [OpenVPN Connect <FiExternalLink/>](https://openvpn.net/client/) o cualquier cliente compatible con OpenVPN para tu dispositivo móvil.
