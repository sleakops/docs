# Cronjobs

## Cronjobs

### Descripción
Un "Cronjob" es una variante especializada de un Job. En lugar de ejecutarse solo una vez, un Cronjob está programado para ejecutarse a intervalos regulares definidos. Por ejemplo, puede configurarse para que se ejecute a diario a las 3 a.m., semanalmente o mensualmente.

### Uso
Para configurar un Cronjob, deberás especificar el comando que se ejecutará y el intervalo de tiempo en el que deseas que se repita. Esta configuración garantiza que el sistema realice la tarea automáticamente sin intervención manual.
