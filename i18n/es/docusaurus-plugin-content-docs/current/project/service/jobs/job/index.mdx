# Jobs

## Jobs

### Descripción
Un "Job" se refiere a un comando simple que se ejecuta en una terminal. Esta tarea se ejecuta una sola vez y realiza una función específica dentro del sistema o la aplicación. A menudo se utiliza para operaciones puntuales como actualizaciones, verificaciones del sistema o tareas de mantenimiento.

### Uso
Para ejecutar un Job, simplemente ingresa el comando deseado en la terminal y espera a que la tarea se complete.
