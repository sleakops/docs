# Gestionando un Node Pool

Aquí encontrarás:

- Cómo actualizar la configuración de un Node Pool.
- Cómo eliminar un Node Pool.

:::info
Recuerda que los Node Pools de build no pueden ser editados ni eliminados.
:::

### 1. Accede a la configuración de tu clúster para ingresar a la sección de Node Pools

Desde la *Lista de Clústeres*, selecciona un Node Pool y accede a la opción de *Configuración*. Luego, haz clic en la caja de *Node Pools*.

### 2. Selecciona el Node Pool

  </div>
</div>

## Cambiar la configuración de un Node Pool

### 1. Haz clic en el botón de configuración en la parte superior derecha de la tarjeta del Node Pool

Actualiza los parámetros en el modal y haz clic en *Guardar*.
