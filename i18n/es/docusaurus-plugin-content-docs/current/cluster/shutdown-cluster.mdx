# Apagar Clúster

:::warning
Para apagar un clúster, asegúrate de que no haya dependencias actualizándose ni procesos de construcción o ejecución activos.
:::

Para apagar un clúster de forma permanente, simplemente haz clic en el botón *Detener* y confirma la acción.

Esta acción solo se puede realizar en clústeres que estén en estado *Activo*.

<img src="/img/cluster/cluster-shutdown-active.png" alt="apagar-cluster-activo" style={{ marginRight: '20px', maxHeight: '200px' }} />

### Encender un clúster

Para encender un clúster, haz clic en el botón *Reproducir* y confirma la acción.

Esta acción está disponible para clústeres en estado *Apagado Programado* y *Apagado*.
