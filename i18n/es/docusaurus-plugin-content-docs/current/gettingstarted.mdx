### Requisitos para Unirse

- Necesitas tener un usuario root en AWS. Este es el usuario inicial creado con permisos completos para gestionar todos los recursos y servicios, y sirve como cuenta principal para AWS Organizations. [Ir a AWS Organizations <FiExternalLink/>](https://console.aws.amazon.com/organizations/v2/home/accounts).
- Necesitas acceso a tus repositorios de código (GitLab, Bitbucket o GitHub).
- Necesitas tener tus servicios en archivos Docker.
- Necesitas poder gestionar tus dominios.
